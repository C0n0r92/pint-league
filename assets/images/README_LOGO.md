# Splash Logo

Please add a `splash_logo.png` file (1024x1024) to this directory.

**Design Guidelines:**
- Beer mug icon with foam
- Orange color (#FF6B35)
- "PINTS LEAGUE" text
- Dark background (#1a1a1a)

For now, you can:
1. Use the SVG file (`splash_logo.svg`) as reference
2. Create the PNG using any design tool (Figma, Canva, etc.)
3. Or use https://www.figma.com to export from the SVG

**Temporary Solution:**
Until you add the logo, the app will use a default Flutter splash screen.
